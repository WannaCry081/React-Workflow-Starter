# Shared Command Rules

Use concise, direct language.
Ask only blocking questions.
Keep scope minimal.
Avoid filler and repetition.

## Workflow Toggle

- Read `.ai/config/workflow-mode.md` before planning or implementation.
- If the file is missing, default to `TDD_MODE: off`.
- `TDD_MODE: off`: do implementation first; tests are optional unless explicitly requested or required by the task.
- `TDD_MODE: on`: do test-first development for new behavior.

## Quality Gate

- Steps are executable.
- Output is clear.
- Stop after required deliverables.
